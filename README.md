# Mechanics of Composite Materials 250.021

This repository contains Python notebooks and class materials from the **Mechanics of Composite Materials** course (250.021) in the Bachelor of Materials Science and Technology program at Montanuniversitaet Leoben.

## Course Overview

The course covers fundamental concepts in composite materials mechanics, including:

- Rule of Mixtures (ROM)
- Solid mechanics problems and stress states
- 3D Hooke's Law
- Practical applications in Python

## Contents

### Notebooks

1. **[PL2 : Rule of Mixtures](Python%20Notebook%20Pratical%20Lecture%202.ipynb)**
   - Introduction to Python for composite materials analysis
   - IDE/Text Editor recommendations
   - Resources for learning
   - Using Jupyter Notebooks
   - Rule of Mixtures (ROM) implementation
   - Practical exercises

2. **[PL3,PL4 : Tensors and 3D Hook Law](Python%20Notebook%20Pratical%20Lecture%203.ipynb)**
   - Review exercises from previous class
   - Tensors in Python
   - Solid mechanics problems (stress state)
   - 3D Hooke's Law applications

## Usage

- Each notebook contains theoretical explanations followed by practical Python implementations
- Run cells sequentially to follow the learning progression
- Modify parameters and observe results to understand concepts better
